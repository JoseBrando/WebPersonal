SIPREL

Siprel es una plataforma de internet para vender recargas electrónicas o saldo de tiempo aire desde celulares, tabletas y computadoras; ganando una comisión por cada compra de saldo.

Con la finalidad de mejorar la experienia del usuario se desarrollo una extensión de siprel.mx para navegadores web. Esta extensión servira como una herramienta para poder realizar recargas electrónicas y pago de servicios.
